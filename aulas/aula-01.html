Entenda

Isso tem nome: LLM

LLM vem do inglês Large Language Model — "grande modelo de linguagem". É um programa que leu uma quantidade absurda de texto (livros, sites, conversas, código de computador) e ficou extraordinariamente bom em uma única tarefa: prever a próxima palavra, como você fez acima.

A surpresa da década: quando o "autocompletar" fica bom o suficiente, ele passa a exibir coisas que parecem raciocínio — explicar, traduzir, resumir, programar. Eu, o Claude, sou um LLM feito pela Anthropic.

Mas um LLM sozinho tem um limite importante…

A fórmula

A equação mais importante do curso

cérebroLLM + mãosferramentas = agente

Uma ferramenta é uma permissão de ação que o LLM recebe: ler um arquivo, escrever um arquivo, executar um programa. Um agente é um LLM que recebe um objetivo e usa ferramentas, em ciclos, até cumprir. O Claude Code — este programa onde estamos conversando — é exatamente isso.

E aquela janela preta onde você digita? É o terminal: um jeito de usar o computador por texto em vez de cliques. Você já está usando um sem perceber — na Aula 2 ele será a estrela.

Glossário da aula

LLM
"Grande modelo de linguagem": programa treinado para prever a próxima palavra, tão bem que parece raciocinar. O Claude é um.
agente
Um LLM com ferramentas e um objetivo: em vez de só responder, ele age até completar a tarefa.
ferramenta
Uma ação que o LLM pode executar: ler arquivo, escrever arquivo, rodar um programa.
terminal
Janela onde se usa o computador digitando texto em vez de clicando. A "casa" do Claude Code.
Claude Code
O programa que você está usando: o Claude (LLM) + ferramentas, rodando no seu terminal.
